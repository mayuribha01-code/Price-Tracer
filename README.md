# Price-Tracer

This Python program creates a simple Amazon product tracker. It sends a request to a product webpage using the Requests library and retrieves the page content. Then it uses Beautiful Soup to parse the HTML and find the product title using the tag id="B0DSBVGKVF". The program prints the title of the product from the webpage.
In this code Sumsung phone price is shown and it's price is giving.

**How to run this code:-**
1. Open terminal
2. install requests (cmd:- pip install requests)
3. install bs4 (cmd:- pip install bs4)
4. run cmd :- python priceTracker
